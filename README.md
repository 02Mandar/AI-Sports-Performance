# AI-Powered Sports Performance Analytics Dashboard

## Overview
This project analyzes cricket and football player statistics using Python, AI-based analytics, and data visualization techniques.

The system evaluates:
- Player performance
- Team strength
- Match trends
- Consistency scores
- Win percentages
- Rankings

## Features
- Sports performance analytics
- AI-generated performance grading
- Team comparison system
- CSV-based dataset analysis
- Player ranking calculation
- Performance score prediction
- Graph visualization

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files
- analysis.py
- player_stats.csv
- requirements.txt
